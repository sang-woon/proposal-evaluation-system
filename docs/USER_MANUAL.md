## 1. 시스템 개요

### 1.1 시스템 소개
본 시스템은 제안서 평가를 위한 웹 기반 평가 시스템입니다. 평가위원이 각 제안서를 평가하고, 관리자가 결과를 집계하여 A4 평가서를 출력할 수 있습니다.

### 1.2 사업 정보
- **사업명**: 경기도의회 블록체인 기반 모바일 의정지원 시스템 구축
- **평가위원**: 8명

### 1.3 평가 체계
| 구분 | 내용 |
|-----|------|
| 평가 항목 | 6개 카테고리, 23개 평가항목 |
| 총점 | 70점 (정성평가) |
| 제안서 수 | 관리자 설정 (2~10개, 기본 7개) |

### 1.4 등급 체계
| 등급 | 의미 | 배점 비율 |
|-----|------|----------|
| 수 | 매우 우수 | 100% |
| 우 | 우수 | 90% |
| 미 | 보통 | 80% |
| 양 | 미흡 | 70% |
| 가 | 매우 미흡 | 60% |

### 1.5 평가 카테고리
1. **전략 및 방법론** (10점) - 4개 항목
2. **기술 및 기능** (20점) - 6개 항목
3. **성능 및 품질** (17점) - 4개 항목
4. **프로젝트 관리** (15점) - 5개 항목
5. **프로젝트 지원** (6점) - 3개 항목
6. **기타** (2점) - 1개 항목

## 2. 평가위원 사용 가이드

### 2.1 시스템 접속
1. 웹 브라우저에서 시스템 URL에 접속합니다
2. 메인 화면에서 **[평가 입력]** 버튼을 클릭합니다

### 2.2 평가위원 로그인
1. 평가위원 이름을 입력합니다
   - 예: "홍길동", "김평가" 등
2. **[평가 시작하기]** 버튼을 클릭합니다

> ⚠️ **주의**: 입력하신 이름으로 평가가 저장됩니다. 오타가 없도록 정확히 입력해주세요.

### 2.3 제안서 선택
- 화면 왼쪽 사이드바에서 평가할 제안서를 선택합니다
- 제안서는 A, B, C, ... 순으로 표시됩니다 (관리자가 설정한 수만큼)
- 각 제안서 버튼 옆에 평가 완료 표시(✓)가 나타납니다
- 상단에 전체 완료율(예: 2/7 완료)이 표시됩니다

### 2.4 평가 입력
1. **카테고리 선택**: 왼쪽 사이드바에서 평가 카테고리를 선택합니다
2. **등급 선택**: 각 평가 항목에 대해 등급(수/우/미/양/가)을 클릭합니다
3. **점수 자동 계산**: 선택한 등급에 따라 점수가 자동 계산됩니다

```
예시) 배점 4점인 항목
- 수: 4.0점 (100%)
- 우: 3.6점 (90%)
- 미: 3.2점 (80%)
- 양: 2.8점 (70%)
- 가: 2.4점 (60%)
```

### 2.5 평가 저장
1. 모든 23개 항목을 평가 완료합니다
2. **[저장]** 버튼을 클릭합니다
3. 저장 완료 메시지가 표시됩니다
4. 자동으로 다음 미평가 제안서로 이동합니다

### 2.6 평가 현황 확인
- 상단의 **[현황]** 버튼을 클릭하면 저장된 평가 목록을 확인할 수 있습니다
- 저장된 평가는 수정이 가능합니다 (제출 전까지)

### 2.7 최종 제출
1. 모든 제안서 평가를 완료합니다
2. **[평가 제출]** 버튼이 활성화됩니다
3. 제출 후에는 수정이 불가능합니다

> ⚠️ **주의**: 제출 후에는 평가를 수정할 수 없습니다. 신중하게 검토 후 제출해주세요.

## 3. 관리자 사용 가이드

### 3.1 관리자 로그인
1. 메인 화면 또는 URL에서 `/admin` 페이지로 접속합니다
2. 관리자 아이디와 비밀번호를 입력합니다
   - 아이디: `admin`
   - 비밀번호: `admin@`
3. **[로그인]** 버튼을 클릭합니다

### 3.2 대시보드 구성
관리자 대시보드는 3개의 탭으로 구성됩니다:

| 탭 | 기능 |
|---|------|
| 📊 평가 현황 | 평가 완료 여부 확인, 점수 집계 |
| 📄 채점표 출력 | 평가위원별 채점표 인쇄 |
| 🏆 총괄표 | 최종 순위표 인쇄 |

### 3.3 평가 현황 탭

#### 평가 완료 여부 확인
- 평가위원 × 제안서 매트릭스 형태로 표시
- ✓: 평가 완료
- -: 미완료
- 각 행/열의 완료율 퍼센트 표시

#### 전체 평가 현황
- 평가위원별 각 제안서 점수 표시
- 제안서별 평균 점수 계산
- 합계 및 통계 정보 표시

#### 제안서별 평가 요약
- 각 제안서의 평가 완료 현황
- 평균 점수 및 개별 평가위원 점수

### 3.4 채점표 출력 탭

#### 필터 옵션
1. **평가위원 선택**
   - 전체 평가위원: 모든 평가위원의 채점표 출력
   - 개별 선택: 특정 평가위원의 채점표만 출력

2. **제안서 선택**
   - 전체 제안서: 모든 제안서에 대한 채점표 출력
   - 개별 선택: 특정 제안서의 채점표만 출력

#### 채점표 구성
- 사업명
- 제안사명, 평가위원명
- 23개 평가항목별 등급 및 점수
- 총점
- 종합의견
- 서명란 (인)

### 3.5 총괄표 탭

#### 총괄표 구성
- 사업명
- 평가일자
- 평가위원 수
- 순위별 제안사명, 평균점수, 평가위원 수
- 평가 미완료 제안서 표시
- 서명란 (평가위원장, 확인자)

### 3.6 테스트 데이터 관리
테스트 목적으로 샘플 평가 데이터를 생성하거나 삭제할 수 있습니다.

1. URL에서 `/seed` 페이지로 접속합니다
2. **제안사 수 선택**: 드롭다운에서 2~10개 중 선택
3. **[🌱 샘플 데이터 생성]** 버튼 클릭
   - 8명의 평가위원이 선택한 수의 제안서를 평가한 데이터가 생성됩니다
   - 제안서 A가 가장 높은 점수, 마지막 제안서가 가장 낮은 점수로 설정됩니다
4. **[📊 현재 데이터 확인]**: 저장된 데이터 상태 확인
5. **[🗑️ 모든 데이터 삭제]**: 테스트 데이터 초기화

> ⚠️ **주의**: 이 페이지는 테스트용입니다. 실제 평가 데이터가 있는 경우 삭제하지 않도록 주의하세요.

## 4. 인쇄 가이드

### 4.1 채점표 인쇄
1. 관리자 대시보드 > **[📄 채점표 출력]** 탭 선택
2. 평가위원 및 제안서 필터 설정
3. **[🖨️ 인쇄하기]** 버튼 클릭
4. 브라우저 인쇄 대화상자에서 설정:
   - 용지 크기: A4
   - 방향: 세로
   - 여백: 기본값 또는 최소
   - 배경 그래픽: 체크 (테두리 인쇄용)

### 4.2 총괄표 인쇄
1. 관리자 대시보드 > **[🏆 총괄표]** 탭 선택
2. **[🖨️ 인쇄하기]** 버튼 클릭
3. 위와 동일한 인쇄 설정 적용

### 4.3 인쇄 팁
- **Chrome 브라우저** 권장 (최적의 인쇄 품질)
- PDF로 저장: 인쇄 대화상자에서 "PDF로 저장" 선택
- 각 채점표는 자동으로 페이지 분리됨 (평가위원당 제안서당 1페이지)

### 4.4 인쇄 시 페이지 구성
```
[채점표 출력 시]
- 평가위원 A의 제안서 1 채점표 (1페이지)
- 평가위원 A의 제안서 2 채점표 (2페이지)
- ...
- 평가위원 B의 제안서 1 채점표 (N페이지)
- ...

[총괄표 출력 시]
- 전체 순위표 (1페이지)
```

## 5. 자주 묻는 질문

### Q1. 평가를 저장했는데 다시 수정할 수 있나요?
**A**: 네, 최종 제출 전까지는 언제든지 수정 가능합니다. 해당 제안서를 선택하고 등급을 변경한 후 다시 저장하면 됩니다.

### Q2. 실수로 잘못된 이름으로 로그인했어요.
**A**: 페이지를 새로고침하고 올바른 이름으로 다시 로그인하세요. 이전에 잘못된 이름으로 저장된 데이터가 있다면 관리자에게 문의하세요.

### Q3. 저장 버튼이 비활성화되어 있어요.
**A**: 모든 23개 평가 항목에 등급을 선택해야 저장 버튼이 활성화됩니다. 미평가 항목이 있는지 확인하세요.

### Q4. 인쇄할 때 레이아웃이 깨져요.
**A**:
- Chrome 브라우저 사용을 권장합니다
- 인쇄 설정에서 "배경 그래픽" 옵션을 체크하세요
- 여백을 "기본값" 또는 "최소"로 설정하세요

### Q5. 관리자 비밀번호를 잊어버렸어요.
**A**: 기본 비밀번호는 `admin@`입니다. 보안상 운영 환경에서는 변경을 권장합니다.

### Q6. 평가 데이터가 사라졌어요.
**A**:
- 같은 브라우저, 같은 컴퓨터에서 접속하세요 (데이터는 브라우저에 저장됨)
- 브라우저 캐시/데이터를 삭제하면 데이터가 사라질 수 있습니다
- 중요한 평가는 주기적으로 인쇄하여 백업하세요

### Q7. 여러 명이 동시에 평가해도 되나요?
**A**: 네, 각 평가위원은 자신의 이름으로 로그인하여 독립적으로 평가할 수 있습니다. 단, 같은 브라우저에서 여러 명이 작업할 경우 각자 다른 탭을 사용하거나, 한 명씩 순차적으로 평가하세요.
